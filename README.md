# LUXIVE E-Commerce Project

This project is built using the "Smart Professional" stack:
- **Frontend**: Next.js (App Router) + Tailwind CSS
- **Backend**: MedusaJS
- **Database**: PostgreSQL
- **Payments**: Razorpay / Stripe

## Project Structure
- `/storefront`: The Next.js frontend application.
- `/backend`: The MedusaJS backend application.

## Getting Started

### Prerequisites
- Node.js (v18+)
- PostgreSQL (Local or Hosted on Neon/Supabase)

### Setup

1. **Frontend**:
   Navigate to `storefront` and run:
   ```bash
   cd storefront
   npm run dev
   ```

2. **Backend**:
   Navigate to `backend` and run:
   ```bash
   cd backend
   npm run dev
   ```
   *Note: You need a running PostgreSQL database for the backend.*
